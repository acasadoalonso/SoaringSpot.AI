# Club Class Handicap List

**Source:** [SC3AH_2026 — IGC Procedures for Handicapped Classes](https://www.fai.org/sites/default/files/document/file/SC3AH_2026.pdf) (Class D gliders, incl. Class DM motorgliders)
**Edition valid from:** 15 April 2026
**Appendix 1 — IGC Club Class List dated:** 1 October 2024

> To be used in conjunction with FAI Sporting Code Section 3, Annex A.

## Definitions

| Term | Meaning |
|------|---------|
| **TCDS** | Type Certificate Data Sheet from the country of registration, the country of manufacture, or EASA |
| **RM** | IGC Reference Mass, from Appendix 1 |
| **MTOM** | Maximum Takeoff Mass allowed. To receive a score, the takeoff mass must be ≤ MTOM (see §1.5) |
| **Hmin** | Minimum Handicap = **0.988** (without adjustments — see §1.6.3) |
| **Hmax** | Maximum Handicap = **1.080** (without adjustments — see §1.6.3) |

## Handicap adjustments (summary)

- **Mass (§1.6.1):** If takeoff mass > RM, handicap increases by **0.004** per 10 kg (or part thereof) above RM. If takeoff mass < RM, handicap is reduced by **0.003** per whole 10 kg below RM, capped at a **0.006** reduction (i.e. ≥20 kg below RM → −0.006).
- **Winglets (§1.6.2):** Adding winglets to a glider originally built/certified without them increases the handicap by **0.004**. Gliders originally certified with winglets only are marked in the list.
- The IGC Reference Mass (RM) is the basis for the mass adjustments. Compliance with certified MTOM (or MTOM without water per TCDS) is mandatory.
